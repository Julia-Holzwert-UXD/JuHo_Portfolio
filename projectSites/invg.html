INVG App Redesign

mobile-app | public-transport | visual-styles

Private Project from 2023/24 for 2 Weeks

Summary

The mobile INVG application gives users quick oversight over their transportational needs. Taking away the stress with key elements in the interface to convey users of punctionality in brief seconds.

Challenge

The challenge was to apply strong visual styles without compromising usability or functionality. Each style required a different approach to layout, interaction, and visual hierarchy.

Approach

I deconstructed the original app to understand its structure and user flow. Using that foundation, I created three redesigned versions that applied the unique visual principles of each selected style while keeping the interface clear and accessible.
